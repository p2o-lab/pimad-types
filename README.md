# PUBLISH

## do-it-yourself 
:exclamation: Sonatype-Nexus server is part of the PLT/SVT-Network! 

First of all: login to npm repository manager system.
```shell script
npm login
```
Then publish ... the tag is 'latest' then.
```shell script
npm publish
```
Publish with specific tag:
```shell script
npm publish --tag <tag>
```
