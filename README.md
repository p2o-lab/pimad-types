# PiMAd-types

This Package contains type definitions for PiMAd-core.

# PUBLISH

First of all: login to npm repository manager system.
```shell script
npm login
```
Then publish ... the tag is 'latest' then.
```shell script
npm publish
```
Publish with specific tag:
```shell script
npm version prerelease --preid=<tag>
npm publish --tag <tag>
```
